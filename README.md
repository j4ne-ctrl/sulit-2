# LANGKAH KERJA PROGRAM PENJUMLAHAN MATRIKS
Program ini berfungsi untuk menjumlahkan dua buah matriks berukuran sama yang dimasukkan oleh user.

Dalam program ini, user diminta untuk memasukkan jumlah baris dan kolom matriks. Setelah itu, user mengisi elemen elemen dari matriks A dan B satu per satu. Program menyimpan data itu ke dalam array dua dimensi. Setalah kedua matriks terisi, program menjumlahkan setiap eleman yang memiliki posisi sama dari matriks A dan B, lalu menyimpannya ke matriks hasil. Lalu program menampilkan hasil penjumlahan dari matriks A dan B di outputnya.
